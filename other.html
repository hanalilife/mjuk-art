Mako 彩妝工作室（網站優化專案）

為既有品牌網站進行架構與視覺優化，提升資訊理解效率與整體使用體驗。

首頁

問題分析

  • 原網站內容分散，缺乏明確的視覺層級
  • 首頁焦點不集中，使用者難以快速理解服務重點
  • 品牌視覺元素不一致，整體辨識度不足

優化重點

  • 重新梳理版面結構，建立清晰的視覺層級與閱讀動線
  • 將分散內容整合為清晰的作品分類，提高瀏覽效率
  • 改用低飽和奶茶色，營造溫柔且專業的品牌氛圍

服務項目區

問題分析

  • 服務內容以長段文字呈現，閱讀負擔較高
  • 各項服務內容方案並未明確分類

優化重點

  • 簡化文字內容，並搭配作品圖片提升資訊可讀性
  • 建立清晰的服務分類系統，讓使用者可快速定位內容

導覽與使用流程

問題分析

  • 導覽層級不清，使用者需多次滾動才能找到目標內容
  • 操作流程缺乏引導，影響整體使用體驗

優化重點

  • 重整導覽結構，降低使用者認知負擔
  • 建立清晰的操作路徑與內容分區，提升操作直覺性

設計決策

  • 採用低飽和暖色系（奶茶色）作為主色，營造溫柔且專業的品牌氛圍
  • 透過留白與區塊化設計，強化內容層級與閱讀節奏
  • 以模組化版面提升內容彈性，方便後續擴充與維護

專案成果

  • 提升整體視覺一致性與品牌辨識度
  • 改善使用者瀏覽流暢度與資訊理解效率

技術與實作

  • HTML / CSS / JavaScript
  • UI / UX 規劃與視覺設計（Figma）
